This project is a simple Image Processing Tool built using Python, OpenCV, and Matplotlib.
The purpose of this project is to learn the basic image preprocessing techniques used in Computer Vision. These techniques are commonly performed before training Artificial Intelligence (AI) and Machine Learning (ML) models.
In this project, a road image is taken as input and different image processing operations are applied to it.

# The project performs the following operations:
Read an image
Resize the image
Crop a selected region
Rotate the image
Flip the image
Blur the image
Sharpen the image
Display all processed images
# Technologies Used
Python
OpenCV
Matplotlib
NumPy
# The program displays:
Original Image
Resized Image
Cropped Image
Rotated Image
Flipped Image
Blurred Image
Sharpened Image
